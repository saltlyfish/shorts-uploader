Shorts Uploader

A personal, single-user command-line tool that uploads my own original short educational videos to my own YouTube channel.

This tool is not a public service and has exactly one user: the owner of the YouTube channel it uploads to. It runs locally on the owner's computer.

What it does

  • Uploads original, self-produced business-education videos (60-second case studies) to the owner's YouTube channel via the official YouTube Data API (videos.insert).
  • Reads the owner's own channel statistics via the YouTube Data API and YouTube Analytics API to improve future videos.
  • Stores its Google OAuth token locally on the owner's computer. Nothing is collected from, shown to, or shared with anyone else.

YouTube API Services

Shorts Uploader uses YouTube API Services. By using it, its user agrees to the YouTube Terms of Service. The Google Privacy Policy applies to Google's handling of data.
