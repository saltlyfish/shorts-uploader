Privacy Policy

Last updated: 19 July 2026

Shorts Uploader is a personal command-line tool operated and used by a single person — the owner of the YouTube channel it uploads to. It has no other users, no accounts, no website login, and no public functionality.

Use of YouTube API Services

Shorts Uploader uses YouTube API Services to upload videos and read channel statistics. Users of this tool agree to the YouTube Terms of Service. Google's handling of your data is described in the Google Privacy Policy.

What data the tool accesses

  • Google OAuth tokens for the owner's Google account, used to authorize uploads and analytics reads.
  • The owner's own channel data: video metadata it uploads, and the channel's own statistics (views, watch time, engagement) via the YouTube Data API and YouTube Analytics API.

How data is stored and shared

  • OAuth tokens are stored only in a local file on the owner's computer. They are never transmitted anywhere except directly to Google's APIs.
  • Channel statistics are stored in a local database on the owner's computer.
  • The tool does not collect, store, process, or share any data of any third party. It serves no web pages, sets no cookies, and uses no analytics or advertising services.
  • No data is sold or shared with anyone.

Data retention and deletion

All stored data (tokens, local statistics) can be deleted at any time by deleting the tool's local files. To request deletion of any data or to ask any privacy question, contact jlai03111@gmail.com; requests are honored within 7 days.

Revoking access

You can revoke this tool's access to your Google account at any time via the Google security settings page (security.google.com/settings/security/permissions).
